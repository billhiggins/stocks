# Higgins Stock AI Project

Father–son project to build an AI-assisted trading system using:
- Supabase (database)
- Python ingestion scripts
- Prediction engines
- ML models
- Modular architecture

This repo contains:
- `ingestion/` — data ingestion scripts
- `models/` — predictive models
- `utils/` — shared helper modules
- `notebooks/` — experimentation notebooks
- `config/` — local config (ignored by Git)
